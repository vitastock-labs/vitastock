# VitaStock - Pharmacy Inventory & Intelligence Platform

## Project Overview

VitaStock is a real-time pharmacy inventory management system that helps pharmacies manage stock faster and with less stress — while automatically preventing errors, expiry, and stockouts.”

## Key Features

- **Real-time Inventory Tracking**: Monitor stock levels as they change.
- **Expiry Date Monitoring**: Get alerts before medications expire.
- **Low-stock Alerts**: Receive notifications when it's time to restock.
- **Workspace-based Management**: Organize inventory by pharmacy location or department.
- **Actionable Insights**: Data-driven intelligence for better procurement.

## Technology Stack

- **Frontend**: Vite, React, TypeScript
- **Styling**: Tailwind CSS v4, shadcn/ui
- **Icons**: Lucide React
- **Package Manager**: pnpm

## Development

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd vitastock

# Step 3: Install dependencies
pnpm install

# Step 4: Start the development server
pnpm run dev
```
